## Available Scripts

## For back-end
Run following command:
1- cd back-end
2- npm install
3- nodemon or npm start


## For front-end
Run following command:
1- cd front-end
2- npm install
3- npm start

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
